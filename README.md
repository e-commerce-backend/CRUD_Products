Para poder consumir la api  
1-Deben instalar el archivo requirements que esta en la carpeta 

2-luego posicionarse en la carpeta products para poder correr el servidor con "fastapi dev main.py --reload" si no estan posicionados en esa carpeta , dará error.

3-Luego de correr el servidor pueden consumir las apis 

De igual forma aqui les dejo la url del servidor local para cuando lo corran:
http://127.0.0.1:8000

Y la url para poder leer la documentacion de la api , está todo muy claro
Documentation at http://127.0.0.1:8000/docs
